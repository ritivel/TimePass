## Chosen Direction: Genda Pop Daily

Core palette:
- Ink: `#141416`
- White: `#FFFFFF`
- Genda: `#F2B33D`
- Warm background: `#FFF8E8`
- Warm tile: `#F5F1E8`
- Muted text: `#817B72`
- Utility teal: `#12A88A`
- Live coral: `#F06A4D`
- Link blue: `#2F7DE1`

Reason:
- It keeps the Monogram-like calm shell but makes TimePass feel warmer and more Indian-daily.
- Genda connects to panchang, festivals, morning utility, and auspiciousness without using tricolor cliches.
- It is more ownable than red, orange, green, or blue lanes already crowded by Indian consumer apps.

Usage:
- Ink is the main chrome and mic color.
- Genda appears in identity moments, selected chips, microphone rays, and warm daily cards.
- Red/coral is only for live urgency or alerts.
- Teal is only for positive/system status.
- Blue is only for links or external actions.
